Es muss nichts zusätzlich instaliert werden, alle pakete die das Program braucht sind im "Chess" Ordner enthalten
Damit das Program läuft muss der Pfad des "Chess" Ordners in der "path.go" Datei im "path" Ordner festgeschrieben werden. 
Dokumentation kann unter "resources\documentation" gefunden werden
Das Program funktioniert vermutlich nur unter Windows, Linux ungetestet --> liegt am "gfxw" Paket welches wahrscheinlich nur für Windows funktioniert
